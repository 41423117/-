# 永恒領域遊戲

一個包含多種遊戲模式的網頁遊戲平台，包含飛鳥射擊遊戲和生存射擊遊戲。

## 功能特色

### 飛鳥射擊遊戲
- 四種難度模式：簡單、一般、困難、隨機
- 炸彈機制：一般、困難和隨機模式會有炸彈
- 歷史記錄系統
- 語言切換：繁體中文、簡體中文、英文
- 自定義背景更換

### 生存射擊遊戲
- 第三人稱射擊遊戲
- Boss戰系統
- 能量塊升級系統
- 永久屬性提升

## 遊戲控制

### 飛鳥射擊遊戲
- 滑鼠：瞄準和射擊
- ESC：暫停遊戲
- 空格鍵：主選單開始遊戲

### 生存射擊遊戲
- WASD/方向鍵：移動
- 滑鼠：瞄準和射擊
- ESC：暫停遊戲
- R：裝填彈藥（無限子彈）
- 空格鍵：射擊

## 安裝與使用

1. 直接訪問 GitHub Pages 網址
2. 或下載檔案後用瀏覽器打開 `index.html`

## 檔案結構

### 技術特色

- 純前端實現，無需後端
- 使用 Canvas 2D 繪圖
- localStorage 儲存遊戲記錄和設定
- 響應式設計
- 模組化 JavaScript 架構
